# منصة رقيم - Raqeem Platform

موقع إلكتروني لمنصة رقيم - مبادرة لنشر الوعي بالتقنية وتمكين الشباب بالمهارات الرقمية.

## الملفات المتضمنة

### الملفات الأساسية (React Build - جاهز للنشر)
- `index.html` - الصفحة الرئيسية المبنية بـ React
- `assets/` - مجلد يحتوي على ملفات CSS و JavaScript المحسنة والصور
- `favicon.ico` - أيقونة الموقع

### ملفات إضافية للتخصيص
- `css/style.css` - ملف CSS منفصل وقابل للتعديل
- `README.md` - هذا الملف

## طريقة النشر على GitHub Pages

### الطريقة الأولى: نشر مباشر
1. قم بإنشاء مستودع جديد على GitHub
2. ارفع جميع الملفات إلى المستودع
3. اذهب إلى Settings > Pages
4. اختر Source: Deploy from a branch
5. اختر Branch: main (أو master)
6. اختر Folder: / (root)
7. احفظ الإعدادات

### الطريقة الثانية: استخدام مجلد docs
1. أنشئ مجلد `docs` في مستودعك
2. انسخ جميع الملفات إلى مجلد `docs`
3. في إعدادات GitHub Pages، اختر Folder: /docs

## التخصيص والتعديل

### تعديل الألوان
يمكنك تعديل الألوان في ملف `css/style.css` من خلال متغيرات CSS:
```css
:root {
  --primary-purple: #4c1d95;
  --primary-blue: #1e40af;
  --accent-cyan: #0891b2;
  --light-cyan: #06b6d4;
}
```

### تعديل المحتوى
- لتعديل النصوص: عدل ملف `index.html`
- لتعديل التصميم: عدل ملف `css/style.css`
- لإضافة صور جديدة: ضعها في مجلد `assets/`
