# موقع مبادرة رقيم للتمكين الرقمي

## نظرة عامة
موقع إلكتروني متجاوب بالكامل باللغة العربية لمبادرة رقيم للتمكين الرقمي، مبني باستخدام HTML5، CSS3، وJavaScript الخالص.

## الملفات المطلوبة
- `index.html` - الملف الرئيسي للموقع
- `styles.css` - ملف التنسيقات والتصميم
- `script.js` - ملف JavaScript للتفاعلات
- الصور المطلوبة:
  - `raqeem_logo.png` - شعار مبادرة رقيم
  - `aabu_logo.jpg` - شعار جامعة آل البيت
  - `yarmouk_logo.png` - شعار جامعة اليرموك
  - `aum_logo.png` - شعار الجمعية الأمريكية في مادبا
  - `cpf_logo.jpg` - شعار مؤسسة ولي العهد
  - `nahno_logo.png` - شعار منصة نحن
  - `workshop_1.jpg` إلى `workshop_6.jpg` - صور ورش العمل

## المميزات
✅ تصميم متجاوب (Responsive Design)
✅ تخطيط RTL للغة العربية
✅ نظام ألوان حديث (بنفسجي، أزرق فاتح، برتقالي)
✅ رسوم متحركة سلسة
✅ نموذج تسجيل تفاعلي مع التحقق من صحة البيانات
✅ شريط تنقل ثابت مع تمرير سلس
✅ معرض صور تفاعلي
✅ تأثيرات hover جميلة
✅ خط Tajawal العربي

## الأقسام
1. **قسم البطل (Hero)** - عنوان رئيسي مع أزرار دعوة للعمل
2. **من نحن** - وصف المبادرة
3. **إحصائياتنا** - أرقام المبادرة مع عدادات متحركة
4. **خدماتنا** - الخدمات الأربع الرئيسية
5. **شركاؤنا** - شعارات الشركاء
6. **معرض الصور** - صور ورش العمل
7. **تواصل معنا** - نموذج التسجيل ومعلومات التواصل

## تعليمات النشر

### النشر المحلي
1. ضع جميع الملفات في مجلد واحد
2. تأكد من وجود جميع الصور في نفس المجلد
3. افتح `index.html` في المتصفح

### النشر على الإنترنت
1. ارفع جميع الملفات إلى خادم الويب
2. تأكد من أن `index.html` هو الملف الرئيسي
3. تحقق من أن جميع الصور متاحة ومرفوعة

### خدمات النشر المجانية المقترحة
- **Netlify**: سحب وإفلات الملفات
- **Vercel**: ربط مع GitHub
- **GitHub Pages**: نشر مباشر من المستودع
- **Firebase Hosting**: خدمة Google المجانية

## التخصيص
يمكن تخصيص الموقع بسهولة من خلال:
- تعديل الألوان في `:root` في ملف CSS
- تغيير النصوص في ملف HTML
- إضافة أو تعديل الصور
- تعديل معلومات التواصل

## الدعم الفني
الموقع مبني بتقنيات ويب قياسية ويعمل على جميع المتصفحات الحديثة:
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## الأمان
- لا يحتوي على مكتبات خارجية
- جميع الأكواد محلية
- نموذج التسجيل يحتاج إلى ربط بخادم لحفظ البيانات

## ملاحظات مهمة
- رابط تيليجرام يحتاج إلى تحديث برابط المجموعة الفعلي
- نموذج التسجيل يعرض رسالة نجاح محلياً، يحتاج إلى ربط بخادم لحفظ البيانات الفعلية
- جميع الصور محسنة للويب وسريعة التحميل
